created a todo application using HTML, CSS, VanillaJS.

User can create task for the day. user can add, edit, delete
and mark the task as done or undone.

Task will persist till the session even if we'll refresh the browser

To Run the application and check out it's feature please follow below steps :-

git clone 
save it in my laptop
double click on html file
application will start in browser
add some tasks and check it's features
